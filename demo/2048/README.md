### 控制
**w a s d** 控制移动

### 介绍
* 分为**有动画**和**没动画**两个版本,**没动画的版本**更方便看逻辑层的代码实现;
* 循环用的是刚刚有所了解的`underscore`,动画实现用的是`css3`,在chorme浏览器下应该运行效果最好.

> 编程学习时间不长,都是些三脚猫功夫,故代码实现方面应该还有许多优化的地方,不过也算是自己瞎鼓捣的,没有参照网上其他2048的代码.

> 用sass生成的css,故每个文件夹会有些sass相关的多余文件.
